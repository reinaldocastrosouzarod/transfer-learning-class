# Prompt da Fase 1 – Scaffolding

## Título
Prompt de scaffolding da fase 1

## Objetivo da fase
Criar a fundação do projeto, incluindo:
- Árvore de pastas do projeto;
- Arquivos‑base de contexto e instrução;
- README inicial;
- requirements.txt inicial;
- .gitignore básico.

## Prompt original (integral)

```
2. mostre o conteúdo completo de cada arquivo criado;
3. resuma em 5-10 linhas o que ficou pronto;
4. diga explicitamente que a fundação do projeto está pronta para a próxima fase.iente inicial do projeto, criando:
1. a árvore de pastas do projeto;
2. os arquivos-base de contexto e instrução;
3. um README inicial;
4. um requirements.txt inicial;
5. um .gitignore básico.
NÃO crie ainda os slides completos, notebooks finais ou práticas completas.
Nesta fase, quero somente a fundação do projeto.
CONTEXTO DO PROJETO
```

## Observação
Este prompt foi executado integralmente na **Fase 1** e resultou na criação da estrutura de diretórios e dos arquivos iniciais listados acima.
