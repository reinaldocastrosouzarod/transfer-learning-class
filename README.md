# Transfer Learning Class Project

## Overview

Este repositório contém a fundação da aula de **Transfer Learning** de 2 horas para alunos de graduação em Ciência de Dados da Engenharia de Produção da PUC‑Rio. O objetivo é estruturar o material de teoria e prática conforme o livro *Introduction to Transfer Learning: Algorithms and Practice* (Wang & Chen, 2023).

## Estrutura de Diretórios

```
transfer-learning-class/
├── src/            # Código‑fonte da aula (ex.: scripts auxiliares)
├── docs/           # Documentação da aula
├── data/           # Datasets abertos e exemplos
├── notebooks/      # Notebooks Jupyter com práticas
├── slides/         # Slides da parte teórica
├── tests/          # Testes (se houver)
├── context.txt     # Descrição de contexto do projeto
├── instruction.txt # Instruções de desenvolvimento
├── requirements.txt
└── .gitignore
```

## Próximas Etapas

- Implementar a lógica da aula em `src/`.
- Criar notebooks de prática em `notebooks/`.
- Desenvolver slides teóricos em `slides/`.
